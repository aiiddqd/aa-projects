# aa-projects

Репозиторий проекта Ads Art Group для разработки и развития сайта, маркетинговой системы и сопутствующей агентной инфраструктуры.

staging: https://aiiddqd.github.io/aa-projects/

## Что это за проект

- Это рабочая база Ads Art Group для запуска и развития сайта по направлению специального и промышленного брендирования.
- В репозитории объединены три слоя:
	- Продуктовый: сайт и UI на Astro.
	- Маркетинговый: позиционирование, сообщения, ограничения и правила кампаний.
	- Операционный: RFC, агентные инструкции, safety и процессы для команды.


## Roadmap

- Этап 1. MVP (текущий фокус)
	- Собрать базовые страницы: Главная, Услуги, Проекты, Процесс, Команда, Контакты.
	- Упаковать ценностное предложение и кейсы под целевые сегменты.
	- Запустить первичную SEO-структуру и формы захвата брифа/заявки.
	- Подготовить аналитику для оценки конверсий и качества обращений.
- Этап 2. Усиление после MVP
	- Расширение функционала сайта (CMS, мультиязычность, SEO-кластеры).
	- Рост контентной и кейсовой базы, доработка страниц под новые сегменты.
	- Оптимизация конверсии на основе данных и запуск регулярных экспериментов.

## Быстрый старт (Astro)

```bash
cd astro
npm install
npm run dev
```

Примечание: по правилам репозитория для фонового запуска можно использовать `astro dev --background`.

## Важные файлы

- [MARKETING.md](MARKETING.md) - основной маркетинговый контекст и ограничения.
- [AGENTS.md](AGENTS.md) - репозиторные правила работы агентов.
- [rfc/260628-mvp.md](rfc/260628-mvp.md) - базовая RFC-спецификация MVP.
- [companion/MESSAGING.MD](companion/MESSAGING.MD) - правила по тону и сообщениям.
- [companion/SAFETY.MD](companion/SAFETY.MD) - safety-ограничения.

## Дополнительно

- Папка `.agents/` содержит локальную конфигурацию агентной среды (скиллы, задачи, инструкции).
- `skills-lock.json` фиксирует подключенные skills/инструменты.
